== Description ==

ConvertKit is an email marketing platform for capturing leads from your WordPress blog. You can create unique forms for each post, then easily create email courses to drip out to your subscribers.

This plugin integrates ConvertKit.com accounts with your WordPress blog.

== Installation ==

1. Upload the `ConvertKit_WordPress` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the ConvertKit API key under the settings tab. Then copy your key from the ConvertKit.com account screen
4. Select a default form to appear on all posts
5. If desired, override the default form on each individual post by editing that post
